# SNT Website

An e-commerce website built with **Next.js** and **Tailwind CSS**.  
It showcases products like Nattu Sakkarai Powder with clean UI and responsive design.

## 🚀 Live Demo
[View on Vercel](https://snt-website.vercel.app)

## 📦 Getting Started
Clone the repository and install dependencies:

```bash
git clone https://github.com/Kishan-ui20/snt-website.git
cd snt-website
npm install
npm run dev
